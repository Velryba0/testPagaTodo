## Test PagaTodo

* Se utilizó Expo, para trabajar con react native.
* Se utilizó el plugin "inline-dotenv", para manejar las variables de estado.
* Se utilizó Firestore, para la parte de base de datos.
* Se utilizó AsyncStorage, para guardar la información de entrar por primera vez.
* Decidí utilizar un custom hook, para realizar la validación de entrar por primera vez, y devolver la información, ya sea desde la api, o desde la base de datos, y almacenarla en el estado local.
